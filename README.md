Template repository for final assignment of basic backend.

## 各種ツールの使い方

`pip install -r requirements.txt` することでインストールできます。

VSCode 以外の方は以下のコマンドを実行してください。

### Flake8

linter です。

```
$ flake8
```

### black

formatter です。

```
$ black .
```

### isort

import 文を並び替える formatter です。

```
$ isort .
```
